# CSIMD
SIMD Accelerated Operations for Complex Numbers.

# Goal
This project aims to provide PyTorch's CPU backend with SIMD support for complex operations.

following operations is implemented:

## AVX
- [x] copy
- [x] fill

## AVX2
- [x] cdiv
- [x] divs
- [x] cmul
- [x] muls
- [x] cadd
